# Theme Customization

## Overview


The system supports custom themes with logo uploads, color schemes, and branding.

## Features
- Primary and secondary color customization
- Logo upload
- Favicon customization
- Light/dark mode support
- Per-user theme preferences

## Implementation
See `/var/www/event-manager/frontend/src/contexts/ThemeContext.tsx`

## Database
Theme settings stored in `theme_settings` table

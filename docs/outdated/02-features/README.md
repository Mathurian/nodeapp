## Feature Matrix by Role

### ADMIN (Full Access)
✅ All features available
✅ System settings and configuration
✅ User management
✅ Event management
✅ Advanced reporting

### ORGANIZER
✅ Event creation and management
✅ Contest and category setup
✅ User assignment
✅ Scoring oversight
✅ Report generation

### JUDGE
✅ View assigned categories
✅ Score contestants
✅ Submit scores for certification
✅ Request uncertification
✅ View contestant bios

### TALLY_MASTER
✅ Certify totals after judges
✅ Review score calculations
✅ Request score removal
✅ Generate tally reports

### AUDITOR
✅ Final score certification
✅ Audit scoring workflow
✅ Approve certifications
✅ Generate audit reports

### BOARD
✅ Oversight and approval
✅ Review finalized results
✅ Access to all reports
✅ Score removal approval

### EMCEE
✅ Access emcee scripts
✅ View contestant information
✅ View judge information
✅ Real-time event tracking

### CONTESTANT
✅ View own scores
✅ View contest information
✅ Update profile
✅ View results

## Installation Options

Event Manager supports multiple installation methods:

### 1. Automated Setup Script (Recommended)
- Zero prerequisites required
- Installs all dependencies automatically
- Configures everything for you
- **Time:** 5-10 minutes

→ **[Start Here: Quick Start Guide](./quick-start.md)**

### 2. Docker Deployment
- Containerized application
- Isolated environment
- Easy scaling
- **Time:** 3-5 minutes

→ **[Docker Setup Guide](./setup-docker.md)**

### 3. Native Installation
- Direct installation on host
- Better performance
- More control
- **Time:** 15-30 minutes

→ **[Native Setup Guide](./setup-native.md)**

## What You'll Need

### Minimum Requirements
- **OS:** Linux, macOS, or Windows with WSL2
- **RAM:** 2GB minimum, 4GB recommended
- **Disk:** 2GB free space
- **Network:** Internet connection for initial setup

### Software (Auto-installed by setup script)
- **Node.js:** Version 18 or higher
- **PostgreSQL:** Version 12 or higher
- **Git:** Any recent version
- **Build Tools:** gcc, make (on Linux)

## First Steps

1. **Choose your installation method** (Automated/Docker/Native)
2. **Follow the appropriate guide** linked above
3. **Access the application** at http://localhost:3001
4. **Login with default credentials:**
   - Email: `admin@eventmanager.com`
   - Password: `password123`
5. **Explore the features** and customize settings

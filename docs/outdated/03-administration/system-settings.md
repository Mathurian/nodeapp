# System Settings

## Overview


Configuration and management of system-wide settings.

## Setting Categories
- General settings
- Security settings
- Email settings
- Backup settings
- Logging settings

## Database Tables
- `system_settings`
- `security_settings`
- `email_settings`
- `backup_settings`

## API
- GET /api/settings - Retrieve settings
- PUT /api/settings - Update settings

See `/var/www/event-manager/src/routes/settingsRoutes.ts`

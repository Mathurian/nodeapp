# Glossary of Terms

## Overview


Definitions of key terms and concepts.

## General Terms

**Event**: Top-level entity representing a competition or contest event (e.g., "2025 State Pageant")

**Contest**: Sub-event within an event (e.g., "Talent", "Interview", "Evening Gown")

**Category**: Judged competition within a contest (e.g., "Vocal Performance", "Stage Presence")

**Criterion**: Individual scoring element within a category (e.g., "Technique", "Creativity")

**Score**: Numerical rating given by a judge for a contestant in a category/criterion

## User Roles

**ADMIN**: System administrator with full access

**ORGANIZER**: Event organizer who manages contests and categories

**BOARD**: Board member with final approval authority

**JUDGE**: Person who scores contestants

**CONTESTANT**: Competition participant

**EMCEE**: Master of ceremonies

**TALLY_MASTER**: Person who aggregates and verifies scores

**AUDITOR**: Person who audits and verifies scoring accuracy

## Workflow Terms

**Assignment**: Linking a judge to a specific category for scoring

**Certification**: Multi-step approval process for scores
- Judge Certification: Judge confirms their scores
- Tally Certification: Tally master verifies calculations
- Auditor Certification: Auditor verifies accuracy
- Board Approval: Final approval

**Deduction**: Points deducted from a contestant's score (requires approval)

**Lock**: Preventing modifications to scores/events

**Release Date**: Date when scores become visible to contestants

## Technical Terms

**JWT**: JSON Web Token - Authentication token format

**RBAC**: Role-Based Access Control - Permission system

**CSRF**: Cross-Site Request Forgery - Security attack type

**ORM**: Object-Relational Mapping - Database abstraction (Prisma)

**WebSocket**: Real-time bidirectional communication (Socket.IO)

**SPA**: Single Page Application - React frontend

**API**: Application Programming Interface - Backend endpoints

**CRUD**: Create, Read, Update, Delete - Basic data operations

**TTL**: Time To Live - Cache expiration time

**Migration**: Database schema change

## Database Terms

**CUID**: Collision-resistant Unique Identifier - Primary key format

**Foreign Key**: Reference to another table's primary key

**Index**: Database performance optimization structure

**Transaction**: Atomic database operation group

**Junction Table**: Table linking two other tables (many-to-many relationship)

## Abbreviations

**TS**: TypeScript

**JS**: JavaScript

**CSS**: Cascading Style Sheets

**HTTP**: Hypertext Transfer Protocol

**HTTPS**: HTTP Secure

**URL**: Uniform Resource Locator

**JSON**: JavaScript Object Notation

**SQL**: Structured Query Language

**CLI**: Command Line Interface

**UI**: User Interface

**UX**: User Experience

**PWA**: Progressive Web App

**E2E**: End-to-End (testing)

**CI/CD**: Continuous Integration/Continuous Deployment
